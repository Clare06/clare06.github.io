Project: Ensuring Children's Safety in School Buses and Schools

Aim:

Design and develop a comprehensive system to enhance the safety of children who travel to school by bus.

Objectives:

  1. Create a system utilizing RFID cards to accurately track children's presence on school buses and within the school premises.
  2. Develop a messaging mechanism to notify parents when their children board the bus and arrive at school.
  3. Implement an alert system to detect any children left behind on the bus after it's parked and alert the driver.
  4. Design a door control mechanism with an ON/OFF switch for controlled access to the bus.

Project Description:

The project aims to enhance the safety and security of children commuting to school by bus. It encompasses the development of a sophisticated system that ensures the well-being of students during their daily journeys. The system employs RFID technology to meticulously monitor the presence of children both on the bus and within the school premises. Timely notifications are sent to parents, offering them reassurance about their children's safety.

My role within the project was the integration of a GSM module with Atmega 32 microcontroller. This integration empowered the system to send messages to parents and drivers, providing real-time updates on the children's whereabouts. My focal contribution was in the domain of C programming, configuring and optimizing the GSM module's functionality to facilitate seamless communication.

Additionally, I played a key role in developing the system's user interface. Leveraging my expertise in PHP, HTML, CSS, and JavaScript, I crafted an intuitive UI that manages the database and supports overall system operations. The interface enhances efficient data management, contributing significantly to the smooth functioning of the safety system.

Project: Route Optimization Application for Enhanced Delivery Efficiency

Description:

Our project revolves around the development of an innovative application tailored to the needs of manufacturing businesses seeking to optimize their order delivery processes. The application focuses on providing companies with intelligently routed maps for delivering goods, taking into account various constraints. By doing so, it significantly enhances operational efficiency and cost-effectiveness, ultimately contributing to enhanced productivity.

Key Features:

  • Efficient Routing: The application's core functionality lies in generating optimized delivery routes that adhere to specific constraints. This feature ensures that goods are delivered with the utmost efficiency and minimal cost.
  • Constraint Consideration: The application takes into consideration various constraints such as delivery time windows, vehicle capacities, and road restrictions. This ensures that the generated routes are not only efficient but also practical.
  • Interactive Mapping: Visual representations of the optimized delivery routes on interactive maps provide users with a clear understanding of the delivery paths and points.
  • User-Focused Dashboard: A user-friendly dashboard is designed for warehouse personnel, allowing them to create orders, manage backhaul operations, and select vehicles for routing. The dashboard also includes profile editing, login, and password recovery functionalities.
  • Enhanced User Management: The application includes robust user management capabilities, ensuring secure user authentication, profile editing, and password recovery.

My Role:

In this project, I played a pivotal role as a team leader and contributor. My responsibilities encompassed the creation of orders, management of backhaul operations, and compiling vehicle lists for efficient routing. I was also actively involved in developing the user-friendly dashboard tailored for warehouse users.

Leveraging my skills in Angular, Spring Boot (REST), and MySQL, I contributed to the implementation of essential features that enabled efficient order and delivery management. Additionally, my role extended to user management, reinforcing the application's security and user authentication aspects.

The Route Optimization Application represents a culmination of our team's efforts to streamline delivery operations, bolstering delivery efficiency, and cost-effectiveness for manufacturing companies.

  • Implemented Public-Private Key Pair Generation: Angular front-end manages secure key generation, with the public key securely stored on the server, ensuring data confidentiality.
  • Client-Side Secure Key Management: Utilized IndexDB within the web application to securely store private keys, safeguarding user-specific encryption.
  • Enhanced Security Measures: Users receive their private keys via email, strengthening security through a verification process before granting access on new browsers or devices.
  • User-Focused Security: Emphasized user security by enforcing strict authentication, allowing login only after verifying the stored private key.

Designed with a strong emphasis on secure encryption practices and user-focused security protocols, ensuring paramount privacy and confidentiality in online communications.
